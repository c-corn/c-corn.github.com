Section Lemmas.


The generalized integral



In this file we extend the definition of integral to allow for arbitrary integration domains (that is, not requiring that the lower endpoint of integration be less or equal than the upper endpoint) and we prove the fundamental properties of the new operator.

Let a, b : IR and assume that F and G are two partial functions continuous in [Min(a,b),Max(a,b)].

Definitions



Before we define the new integral we need to some trivial interval inclusions.

Variables a b : IR.
Hypothesis Hab : Min a b [<=] Max a b.

Lemma compact_inc_Min_lft : forall H, ⊆ (compact (Min a b) a H) (Compact Hab).

Lemma compact_inc_Min_rht : forall H, ⊆ (compact (Min a b) b H) (Compact Hab).

End Lemmas.

Section Definitions.

The integral is defined by the formula ∫abf=∫min(a,b)bf-∫min(a,b)af, inspired by the domain union rule; obviously it coincides with the classical definition, and it collapses to the old one in the case a ≤ b.

Variables a b : IR.
Hypothesis Hab : Min a b [<=] Max a b.
Variable F : PartIR.

Hypothesis HF : Continuous_I Hab F.

Lemma Integral_inc1 : Continuous_I (Min_leEq_lft a b) F.

Lemma Integral_inc2 : Continuous_I (Min_leEq_rht a b) F.

Definition ∫ :=
  ∫I _ _ (Min_leEq_rht a b) F Integral_inc2[-]∫I _ _ (Min_leEq_lft a b) _ Integral_inc1.

Lemma Integral_integral : forall Hab' HF', ∫ [=] ∫I a b Hab' F HF'.

End Definitions.


Section Properties_of_Integral.

Properties of the Integral



All our old properties carry over to this new definition---and some new ones, too. We begin with (strong) extensionality.

Variables a b : IR.
Hypothesis Hab : Min a b [<=] Max a b.
Variables F G : PartIR.

Hypothesis contF : Continuous_I Hab F.
Hypothesis contG : Continuous_I Hab G.

Lemma Integral_strext : ∫ contF [#] ∫ contG ->
 {x : IR | Compact Hab x | forall Hx Hx', Part F x Hx [#] Part G x Hx'}.

Lemma Integral_strext' : forall c d Hcd HF1 HF2,
 ∫ (Hab:=Hab) (F:=F) HF1 [#] ∫ (a:=c) (b:=d) (Hab:=Hcd) (F:=F) HF2 ->
 a [#] c or b [#] d.

Lemma Integral_wd : ≈ (Compact Hab) F G -> ∫ contF [=] ∫ contG.

Lemma Integral_wd' : forall a' b' Ha'b' contF', a [=] a' -> b [=] b' ->
 ∫ contF [=] ∫ (a:=a') (b:=b') (Hab:=Ha'b') (F:=F) contF'.

The integral is a linear operator.

Lemma Integral_const : forall c (H : Continuous_I Hab [-C-]c), ∫ H [=] c[*] (b[-]a).

Lemma Integral_comm_scal : forall c (H : Continuous_I Hab (c{**}F)), ∫ H [=] c[*]∫ contF.

Lemma Integral_plus : forall H : Continuous_I Hab (F{+}G), ∫ H [=] ∫ contF[+]∫ contG.

Lemma Integral_inv : forall H : Continuous_I Hab {--}F, ∫ H [=] [--] (∫ contF).

Lemma Integral_minus : forall H : Continuous_I Hab (F{-}G), ∫ H [=] ∫ contF[-]∫ contG.

Lemma linear_Integral : forall α β (H : Continuous_I Hab (α{**}F{+}beta{**}G)),
 ∫ H [=] α[*]∫ contF[+]β[*]∫ contG.

If the endpoints are equal then the integral vanishes.

Lemma Integral_empty : a [=] b -> ∫ contF [=] 0.

And the norm provides an upper bound for the absolute value of the integral.

Lemma Integral_leEq_norm : AbsIR (∫ contF) [<=] Norm_Funct contF[*]AbsIR (b[-]a).

End Properties_of_Integral.

Section More_Properties.

Two other ways of stating the addition law for domains.

Lemma integral_plus_Integral : forall a b Hab F c Hac Hcb Hab' Hac' Hcb',
 ∫I c b Hcb F Hcb' [=] ∫I a b Hab F Hab'[-]∫I a c Hac F Hac'.

Lemma integral_plus_integral' : forall a b Hab F c Hac Hcb Hab' Hac' Hcb',
 ∫I a c Hac F Hac' [=] ∫I a b Hab F Hab'[-]∫I c b Hcb F Hcb'.

And now we can prove the addition law for domains with our general operator.

Assume c : IR.

Variables a b c : IR.
Hypothesis Hab' : Min a b [<=] Max a b.
Hypothesis Hac' : Min a c [<=] Max a c.
Hypothesis Hcb' : Min c b [<=] Max c b.
Hypothesis Habc' : Min (Min a b) c [<=] Max (Max a b) c.

Variable F : PartIR.

Hypothesis Hab : Continuous_I Hab' F.
Hypothesis Hac : Continuous_I Hac' F.
Hypothesis Hcb : Continuous_I Hcb' F.
Hypothesis Habc : Continuous_I Habc' F.


Lemma Integral_plus_Integral : ∫ Hab [=] ∫ Hac[+]∫ Hcb.

Notice that, unlike in the classical case, an extra hypothesis (the continuity of F in the interval [Min(a,b,c),Max(a,b,c)]) must be assumed.

End More_Properties.

Section Corollaries.

Variables a b : IR.
Hypothesis Hab : Min a b [<=] Max a b.
Variable F : PartIR.

Hypothesis contF : Continuous_I Hab F.

As a corollary, we get the following rule:

Lemma Integral_op : forall Hab' (contF' : Continuous_I (a:=Min b a) (b:=Max b a) Hab' F),
 ∫ contF [=] [--] (∫ contF').



Finally, some miscellaneous results:

Lemma Integral_less_norm : a [#] b -> forall x, Compact Hab x -> forall Hx,
 AbsIR (F x Hx) [<] Norm_Funct contF -> AbsIR (∫ contF) [<] Norm_Funct contF[*]AbsIR (b[-]a).



Lemma ub_Integral : a [#] b -> forall c, (forall x, Compact Hab x -> forall Hx, AbsIR (F x Hx) [<=] c) ->
 forall x, Compact Hab x -> forall Hx, AbsIR (F x Hx) [<] c -> AbsIR (∫ contF) [<] c[*]AbsIR (b[-]a).

End Corollaries.

Lemma Integral_ap_zero : forall a b Hab (F : PartIR) contF, a [#] b -> forall x,
 Compact Hab x -> forall Hx, 0 [<] F x Hx -> (forall x, Compact Hab x -> forall Hx, 0 [<=] F x Hx) ->
 0 [<] AbsIR (∫ (a:=a) (b:=b) (Hab:=Hab) (F:=F) contF).

Lemma Integral_eq_zero : forall a b Hab (F : PartIR) contF x, Compact Hab x ->
 (forall Hx, 0 [<] F x Hx) -> (forall x, Compact Hab x -> forall Hx, 0 [<=] F x Hx) ->
 ∫ (a:=a) (b:=b) (Hab:=Hab) (F:=F) contF [=] 0 -> a [=] b.
